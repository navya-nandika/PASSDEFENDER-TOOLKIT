# PASSDEFENDER-TOOLKIT


-> A modern cybersecurity application project for password strength analysis and targeted custom wordlist generation. 

-> Designed for students, ethical hackers, and security professionals, PassDefender ToolKit offers an intuitive GUI, real-time feedback, advanced attack pattern generation, and export functionality compatible with commonly used pentesting tools.

_Features_

1. Password Strength Analyzer

-> Real-time analysis using zxcvbn (entropy, pattern checks, estimated crack time)

-> Actionable feedback on weaknesses and improvement suggestions

-> Show/Hide toggle for privacy during password entry

2. Custom Wordlist Generator

-> Five user input fields: Name, Pet/Nickname, Birth Year, Favorite Place, Favorite Number

-> Generates hundreds of combinations: case variations, leetspeak, special characters, reversals, permutations

-> Scrollable preview box for instant review

-> Export wordlist as .txt file for tools like hashcat, John the Ripper

3. Dark-Themed GUI

-> Cyber-inspired: Terminal colors, monospace fonts, dual module layout
